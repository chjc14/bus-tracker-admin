About This Project

Jasva Yana is a student-led initiative designed to make the college commute easier and more predictable. This portal provides a real-time, live-tracking solution for all college buses, allowing students to see exactly where their bus is, reduce wait times, and plan their travel more efficiently.

This system is built using modern web technologies, including real-time data synchronization with Firebase and interactive mapping with Leaflet.js.
